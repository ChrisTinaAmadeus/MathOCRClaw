15. (13 分)  
在锐角三角形△ABC中，设角A、B、C所对的边分别为a、b、c，已知$\sin^2 B + \sin^2 C = \sin^2 A + \sin B \sin C$且$a = \sqrt{3}$。  
(1) 求角A的大小；  
(2) 求$b + c$的取值范围。  
【答案：(1) $\dfrac{\pi}{3}$；(2) $(\sqrt{3}, 2\sqrt{3}]$】

17. (15 分)  
已知D、E是曲线$L: \dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1\ (a > 0, b > 0)$与x轴的交点，点$A(0,3)$，B在曲线L上，且点B异于点A、D、E，直线AD与BE相交于点S，直线AE与BD相交于点T。

(2) 若O为坐标原点，直线AB与DE相交于点R，直线ST与DE相交于点H，求证：$|OR|$、$|OE|$、$|OH|$成等比数列。  
【答案：(1) $(0, \pm \sqrt{3})$；(2) 证明过程见手写，结论成立】

18. (17 分)  
已知函数$f(x) = x - \ln x$，$g(x) = x e^x - 1 - x\ (x > 0)$。  
(1) 求曲线$y = g(x)$经过点$(0, -1)$的切线方程；  
(2) 若$f(x) + g(x) \geq a x$对定义域内的任意x恒成立，求实数a的取值范围。  
【答案：(1) $y = -1 + x$；(2) $a \leq 1$】

19. (17 分)  
已知数列$\{a_n\}$满足$a_n = p a_{n-1} + q_n\ (n \in \mathbb{N}^*)$，其中$p, q_n \in (0,1)\ (\forall n \in \mathbb{N}^*)$。  
(1) 若$p_k = q_k = 0.4\ (k \in \mathbb{N}^*)$，证明：对任意的正整数k，都有$a_{k+1} > a_k$；  
(2) 若$p_k + q_k = 1\ (k \in \mathbb{N}^*)$，  
 (i) 试探究是否存在满足条件的$p_1, p_2, p_3$，使得$a_3 \leq 1$，若存在则求出所有可能的$p_i$值，若不存在，请说明理由；  
 (ii) 取$p_n$为$s p_{n-1}$的小数部分$k \in \mathbb{N}^*$，其中$p_1 \in \left\{\dfrac{1}{5}, \dfrac{2}{5}, \dfrac{3}{5}, \dfrac{4}{5}\right\}$，$s \in \{2, 3, 4\}$，  
  ① 由$P_n + Q_n = 1$，$a_n = p_n a_{n-1} + q_n$，得$a_n - 1 = p_n(a_{n-1} - 1)$，  
  ② ∴ $a_3 - 1 = p_1 p_2 p_3 (a_0 - 1) = p_1 p_2 p_3 > 0$，故$a_3 > 1$，不存在满足$a_3 \leq 1$的情形。  
【答案：(1) 证明略；(2)(i) 不存在；(ii) 见手写推导】
