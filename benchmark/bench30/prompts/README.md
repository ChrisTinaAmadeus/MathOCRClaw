# Markdown 输出格式速查（与 extract_v1 一致）

评估器硬性要求摘要。完整规则与模型提示词见同目录 [`extract_v1.txt`](extract_v1.txt)。

## 必须有

| 元素 | 写法 | 作用 |
|------|------|------|
| 大题标题 | `一、选择题` / `二、填空题` / `三、解答题` | 识别题型 |
| 小题题号 | 行首 `1.` `12.` | 切题与对齐 |
| 答案 | `【答案：C】` 或 `___…___` | 算答案准确率 |
| 拒答 | `7. [无法识别的选择题]` | 拒答指标 |

## 推荐有

- 可选首行：`[格式正常]`
- 公式：`$...$`
- 插图：`![插图](figure_12.jpg)`
